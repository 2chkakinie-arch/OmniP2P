# OmniP2P
[ https://omnip2p.netlify.app/ ]
OmniP2Pの公式リポジトリです。最新バージョンである **v3** を提供しています。

## 🚀 クイックスタート

最新バージョンのスクリプトは、以下のCDN（jsDelivr）から直接読み込んで利用できます。

```html
<script src="https://cdn.jsdelivr.net/gh/2chkakinie-arch/OmniP2P@main/v3.js"></script>
```

## 📦 利用方法

上記のURL（ `v2.js` ）をプロジェクトのHTMLファイル、またはJavaScript環境に組み込んで使用してください。
